Don't forget:
C помощью File -> Open можно открыть только проекты, подготовленные ранее в самой IDEA!
Для того , чтобы запусить любой Java-проект клонированный с Git, вам нужно проделать нехитрую последовательность действий по импорту проекта,
чтобы IDEA могла создать свои рабочие файлы:
    В IDEA : File -> New -> Project from Version Control - вставить HTTPS проекта на Git
